🔑 Quick Setup (One-Time Only)

Follow these steps to create your Figma API token:

Step 1
Open Figma Settings
Click your avatar in the top-left, then select "Settings"
Step 2
Go to Security Tab
Find "Personal access tokens" section
Step 3
Name Your Token
Enter a descriptive name (e.g., "Component Radar")
Step 4
Set Expiration
Select "90 days" for longer validity
Step 5
✅ Check "file_content:read"
This is the ONLY required permission
Step 6
Generate Token
Click "Generate token" and copy it immediately
Step 7
Copy Your Token
Save this token securely - you'll need it for Component Radar
Step 7
Go to All Projects
In Figma sidebar, click "All projects"
Step 8
Copy Project Link
Right-click your project → "Copy link" to get Project ID
From URL: figma.com/files/…/PROJECT_ID/…
